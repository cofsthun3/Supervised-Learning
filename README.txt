Title: CS7641 - HW1 Supervised Learning 

**Description**
This project introduces the use of machine learning to test accuracy of scalping by testing the precision using different models.


**Environment**

To build the environment you will first have to install the required packages from the requirements.txt file.

1. First navigate to the src folder in the terminal
2. pip install -r requirements.txt
3. pip install ipykernel
4. python -m ipykernel install --user --name=trade_bots
5. Start jupyter notebook using 'Jupyter Notebook' command in terminal
6. Run through the code sequentially.
